# ☕ Café Cremoso da Vera - Catálogo Digital

Uma landing page profissional e responsiva desenvolvida para o **Café Cremoso da Vera**, focada na apresentação de um cardápio digital de produtos artesanais e na conversão de vendas diretas via WhatsApp.

## 🚀 Visão Geral do Projeto

Este projeto foi construído para fornecer uma presença digital elegante e funcional para um negócio local. O design adota um conceito "rústico escuro" para destacar as imagens dos produtos e transmitir a sensação de qualidade premium, tradição e aconchego.

## 🛠️ Tecnologias e Práticas Utilizadas

* **HTML5 Semântico:** Estruturação clara e lógica do conteúdo para melhor indexação e manutenção.
* **CSS3 Avançado:**
  * Uso de **CSS Grid e Flexbox** para alinhamento dinâmico dos cartões de produtos.
  * **Media Queries** para garantir uma experiência 100% responsiva (Mobile, Tablet e Desktop).
  * **Variáveis Globais (`:root`)** para padronização e fácil manutenção da paleta de cores.
* **Integração com API do WhatsApp:** Links dinâmicos (`wa.me`) gerando mensagens pré-formatadas para otimizar a experiência de compra do cliente.

## ✨ Destaques da Interface

* **Design Responsivo Seguro:** O layout se adapta perfeitamente a smartphones e tablets, com uma restrição máxima de largura (`max-width: 1600px`) para não distorcer em monitores Ultra-Wide.
* **Navegação Sticky:** Menu superior fixo para facilitar a usabilidade enquanto o usuário navega pelo catálogo.
* **Leiturabilidade Otimizada:** Textos de apresentação justificados e com contraste ajustado para leitura confortável em telas escuras.
